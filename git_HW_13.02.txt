
1  Как скопировать на свой компьютер репозиторий из github.com, что должно быть сделано предварительно?	

	На GitHub.com выбрать нужный репозиторий. Над списком файлов нажать на button Code и скопировать URL-адрес репозитория(c SSH).
	В терминале, с помощью команды cd указаваем путь куда хотим клонировать репозиторий
	Вводим git clone и вставляем скопированный URL адрес(c SSH).

2  Как создать git репозиторий из локального каталога на своем компьютере. Как его отправить на github.com?

	git init					            	команда для создания репозитория из локального каталога 
	git add название_файла						команда для добавления конкретного файла
	git commit -m “короткий и ясный комментарий” имя_файла    	коментарий к правке
	или
	git add .					            	команда для добавления всех папок и файлов
	git commit -m “короткий и ясный комментарий”      		коментарий к правке
    
	На GitHub.com выбрать создать репозиторий и не ставить галочку на Add в README file
	После этого мы получим ссылку на репозиторий, которую надо скопировать
	На локальном компьютере, в терминале добавить эту ссылку командой:
	git remote add origin ссылка
	git push origin main				  		команда для отправки файла и изменений в GitHub

3  Как получить на локальный компьютер изменения с github.com

	git pull					            	команда для получения изменений
	cat README.md					        	команда для просмотра изменений

4  Какую информацию нам дают команда :

	git status					          	команда показывает состояние файлов
	git remote -v					        	команда для просмотра связки с удалённым репозиторием
